# README

开发目录。

使用 [vite-plugin-monkey](https://github.com/lisonge/vite-plugin-monkey) 在 vite 中构建。

> [!WARNING]
> 由于我将 `inject-into` 设置为 `page`，所以 Safari + UserScripts 插件的 `GM` API 不可用（TamperMonkey 插件正常）。

## install

```bash
bun install
```

## run

```bash
bun dev
```

## build

```bash
bun build
```
